  <action>
    Create `src/components/hero.tsx` as a client component.

    First line: `"use client"`
    Imports: useRef, useState, useEffect from "react".

    Define a `useMouse` hook inside the file (not exported):
    - Returns `{ x: number, y: number }` state, initialized to `{ x: 0, y: 0 }`.
    - In useEffect, attaches a `mousemove` listener on `window` with `{ passive: true }`.
    - Handler: `(e: MouseEvent) => setPos({ x: e.clientX, y: e.clientY })`.
    - Cleanup removes the listener on unmount.

    Define and export `HeroBackground` as the default-or-named export:

    Inside HeroBackground:
    - Call `useMouse()` to get `{ x, y }`.
    - Compute parallax offset: factor = 0.015 (subtle — < 20px travel).
      On server side (typeof window === "undefined"), treat center as 0.
      dx = (x - window.innerWidth / 2) * factor
      dy = (y - window.innerHeight / 2) * factor
    - Return a `<div>` with `className="absolute inset-0 overflow-hidden pointer-events-none"`.

    Inside, two layers:

    Layer 1 — CSS grid pattern (static):
    - A `<div>` with `className="absolute inset-0"`.
    - `style={{ backgroundImage: "linear-gradient(var(--color-outline-variant) 1px, transparent 1px), linear-gradient(90deg, var(--color-outline-variant) 1px, transparent 1px)", backgroundSize: "40px 40px", opacity: 0.3 }}`.
    - This creates the subtle grid using the outline-variant token (warm grey in both modes).

    Layer 2 — Floating icon layer (parallax-shifted):
    - A `<div>` with inline style `transform: translate3d(${dx.toFixed(2)}px, ${dy.toFixed(2)}px, 0)`
      and `willChange: "transform"` to promote to compositor layer (prevents Chromebook jank, per Pitfall 4).
    - `className="absolute inset-0"`.
    - Inside: 5-7 decorative French-learning themed icons as SVG elements.
      Because lucide-react may not have French-specific icons, use simple outlined geometric
      shapes or letters styled with coral (#e57373 = var(--color-primary-container) in light mode).
      Suggested: render a few large outlined SVG circles/squares with text inside (e.g., "«»", "A", "B", "é")
      positioned at various absolute positions across the hero.
      Use `stroke="var(--color-primary-container)"` and `fill="none"` for the outlined-coral look per D-07.
      Keep icons at opacity 0.5, spread across the 100% x 100% space at scattered positions
      (e.g., top-left, center-right, bottom-center, etc.) using `style={{ position: "absolute", ... }}`.
      Implementer has creative latitude on exact icon shapes and positions per D-07.

    The HeroBackground component should be `export function HeroBackground()`.

    IMPORTANT: HeroBackground renders INSIDE a relatively-positioned container in page.tsx —
    it uses `absolute inset-0` to fill its container. Do not add height here; the parent
    in page.tsx sets the height.

    Also export the disabled CTA button pattern as `export function DisabledCTA()`:
    - A `<div className="relative group inline-block">` wrapper.
    - Inside: a `<button>` with `disabled` attribute, text "Create account",
      className: `px-6 py-3 bg-primary text-on-primary rounded font-label text-sm opacity-60 cursor-not-allowed`
    - Below the button: a tooltip `<span>` with text "Coming soon",
      className: `absolute -top-9 left-1/2 -translate-x-1/2 bg-on-surface text-surface text-xs px-2 py-1 rounded opacity-0 group-hover:opacity-100 transition-opacity whitespace-nowrap pointer-events-none`
    Per D-06: button is visible but disabled. Tooltip on hover. Phase 2 wires it to /signup.

    Do NOT use any color values as raw hex strings in JSX — only CSS custom properties
    via var() or Tailwind utility classes.
  </action>
